Från manuell hantering till automatiserat flöde. Hämta faktureringsunderlag, fördela jämnt, exportera med ett klick.

Problemet idag

Manuellt arbete som tar tid varje dag

100-150 fakturor/mån

Överförs manuellt mellan Blikk och Visma eEkonomi. Varje faktura kräver kopiering av kunduppgifter, artiklar och belopp.

Ingen månadsbeteckning

Fakturor saknar automatisk månadsbeteckning, vilket gör det svårt för kunder att se vilken period fakturan avser.

Ojämn kassaflödesfördelning

Alla fakturor skickas samtidigt, vilket ger toppar i inbetalningar och gör det svårt att planera likviditet.

Webbhotell separat

Webbhotellfakturor hanteras utanför Blikk utan samlad överblick. Risk att faktureringar missas.

Automatisk synk

Faktureringsunderlag hämtas automatiskt från Blikk varje dag. Identifierar nya faktureringstillfällen och skapar lokala fakturor.

Månadsbeteckning

Varje fakturarad får automatiskt aktuell månadstext, t.ex. "Din Marknadskoordinator — Mars 2026".

Jämn kassaflödesfördelning

LPT-algoritm fördelar fakturor jämnt över 4 veckor per månad för stabil och förutsägbar likviditet.

Teknisk lösning

Byggt med beprövad teknik

Integration

  • Blikk API — Hämtar projekt och faktureringsunderlag
  • Visma eEkonomi API — Skapar fakturadrafts via OAuth 2.0
  • Säker tokenhantering med krypterad lagring

Hosting & drift

  • Körs på befintlig Oderland cPanel-hosting
  • Ingen extra driftkostnad
  • Automatisk schemaläggning via cron

Säkerhet

  • OAuth 2.0 med automatisk token-refresh
  • Krypterad lagring av API-nycklar (AES-256)
  • Inloggningsskyddad — enbart för administratör

Dashboard

  • Realtidsöversikt med sammanfattningskort
  • Veckofördelningsdiagram med Chart.js
  • Prognoser och trendanalys
